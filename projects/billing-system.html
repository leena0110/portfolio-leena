RITE ELECTRICALS Billing & Inventory System

Application Type Desktop Application
Storage CSV File System
Business Scale 500+ Daily Transactions
Status Production Ready

Project Overview

The RITE ELECTRICALS Billing System is a full-scale business automation tool designed specifically for electrical goods businesses that:

  • Handles 500+ daily transactions efficiently
  • Manages real-time inventory using FIFO (First-In-First-Out) method
  • Automates GST billing (18% = 9% CGST + 9% SGST)
  • Tracks complete customer histories and credit systems
  • Generates and sends digital receipts via WhatsApp
  • Supports multi-user roles (Admin & User)
  • Exports comprehensive business reports in CSV format

Key Features

Authentication System

Secure role-based access with Admin & User roles. Restricted access for sensitive operations like product rate changes and data deletion.

FIFO Inventory

First-In-First-Out stock management with automated updates on purchase/sale. Real-time stock level alerts and tracking of opening/closing stock.

GST Billing System

Automated GST calculation (18% = 9% CGST + 9% SGST). Supports both retail and wholesale pricing with printable receipts.

WhatsApp Integration

Automated digital receipt delivery via WhatsApp. Customers receive invoices instantly on their mobile devices.

Customer Management

Complete customer database with purchase history tracking. Credit system management and auto-suggest search functionality.

Reporting & Analytics

Daily/Monthly/Custom sales reports. Stock summary analytics with CSV export capability for external analysis.

Data Storage System

The application uses CSV files for data persistence, making it lightweight and portable:

products.csv
Product catalog with SKU, name, rate, GST, stock levels, and category information
customers.csv
Customer database with contact details, purchase history, and credit information
sales_receipts.csv
Complete sales transaction records with timestamps and payment details
purchases.csv
Purchase order records from suppliers with quantity and pricing details
future_rate_changes.csv
Scheduled product rate changes for future effective dates

Dynamic Files: bill_*.csv and receipt_*.txt are generated for each transaction for record-keeping and printing.

Design Patterns Used:

  • MVC Architecture: Clear separation between Models, Views, and Controllers
  • Factory Pattern: For creating standardized UI widgets and components
  • Observer Pattern: For real-time updates between different modules
  • Modular Component Design: Independent, reusable components

Performance Optimizations:

  • Lazy Loading: Data loaded on-demand to reduce initial startup time
  • Caching: Frequently accessed data cached in memory
  • Batch File Operations: Efficient CSV read/write operations
  • Non-blocking UI: Background processing for long operations

WhatsApp Integration: Uses webbrowser API to open WhatsApp Web with pre-filled message containing invoice details for instant delivery to customers.
